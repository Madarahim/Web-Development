Details:

This is a Java Program that I created called, Studentoons. Its essentially an iTunes like MP3 player. You can add (via drag and drop or button clicks) or delete songs, create or delete playlists, shuffle songs, loop songs, change the volume, pause songs, stop songs, skip to the next or previous song, play a recent song, and just about any other thing you can do with a regular MP3 player. This was created using a GUI and it stores the basic information and the path of each music file in a local derby database that was coded using SQL. The individual playlists and their information are also stored in a database.
