Details:

This is a project that I and a group of classmates created, and is a Multi-User Time Capsule Website. Time capsules are essentially containers filled with memorable items, such as pictures, that are closed and reopended at a later date in time so that one may relive precious moments of the past. People can create profiles, create time capsules, have friends, invite friends to contribute to their time capsules, and many other features we are planning to add. It was created using HTML, CSS, JavaScript, and PHP. Currently, it uses a local SQL server to store profile and timecapsule information for users.
