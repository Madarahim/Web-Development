Details:

This is a C++ program that models a company, manages the information of specific employees, and calculates the salary payments the company has to make. I created multiple Header files containing the classes of the different types of employees, such as Faculty, Staff, and Partime. All of these classes are subclasses of the Employee class which contains virtual methods, which are overridden by the child classes via polymorphism, that calculate a specific employee's salary and display their personal information. Each employee has different attributes, such as degree and education. In the main method, I use a pointer array to store all the employees and use simple for-loops to calculate the total salaries that the company has to pay.
