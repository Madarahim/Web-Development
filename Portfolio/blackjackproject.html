Details:

This is a Java Program that I created which lets you play a virtual version of BlackJack. You have a GUI that gives you the options to either create a deck, shuffle a deck, show all cards, or play blackjack against the dealer. Once the deck runs out you must create a new deck. This program was developed using simple Object-Oriented design. There is a Card Class (which has individual card information), Deck Class (which contains 52 Card objects), and a Hand Class (which holds drawn cards for both the player and the dealer).
